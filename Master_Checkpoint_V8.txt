🏆 How The Coaches System Works (NEW v8)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

5-Tier Classification:
─────────────────────────────────────────────────────────────
  Tier                    ATK (dampened)   DEF (dampened)
  Elite Tactician          1.042            0.944
  Seasoned Strategist      1.021            0.965
  Solid Organizer          1.000            0.986
  Developing Manager       0.986            1.014
  Unproven Appointment     0.965            1.042
─────────────────────────────────────────────────────────────

Key Design Principles:
• Defense impact > Attack impact (coaches organize defense more)
• Dampening (0.75x) prevents unrealistic swings
• Tier gap bonus only kicks in when gap >= 2 tiers
• Max combined swing: ~18-22% on lambda → ~8-12% win probability
• Elite Tactician doesn't guarantee wins, but raises floor + ceiling

3-Layer Impact Chain:
─────────────────────────────────────────────────────────────
  LAYER                         EXAMPLE: ARG vs HAITI
  ─────                         ────────────────────

  1. Form + Ranking blend       lam_a = 2.10, lam_b = 0.62
         │
  2. Star player impact         lam_a = 2.18, lam_b = 0.62
         │
  3. Coach ATK multiplier  →    lam_a x 1.042 = 2.27  (Scaloni: Elite)
         │                      lam_b x 0.986 = 0.61  (Developing)
         │
  4. Coach DEF multiplier  →    lam_a x 1.014 = 2.30  (Weak defense)
         │                      lam_b x 0.944 = 0.58  (Scaloni's defense)
         │
  5. Tier gap bonus        →    lam_a x 1.045 = 2.40  (+3 tiers x 0.015)
         │                      lam_b x 1.000 = 0.58
         │
  6. Final                 →    ARG ~2.40 xG vs Haiti ~0.58 xG
─────────────────────────────────────────────────────────────

⚙️ How The Prediction Engine Works (v8 — Coaches Edition)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Two Engines Working Together:

Engine 1: prediction_engine.py (Terminal)
  Dixon-Coles Poisson + Monte Carlo
  → 10,200 simulations
  → ~instant results
  → Used by predict.py, content_automation.py, app.py
  → NOW INCLUDES: Coaches impact in compute_lambdas()

Engine 2: app.py (Streamlit Web)
  Reads from prediction_engine.py predict()
  → Charts, knockout sim, card risk, H2H
  → Coaches Matchup panel
  → WC 2026 Group Stage Simulator
  → Underdog Spotlight

Complete Pipeline (v8):
─────────────────────────────────────────────────────────────
  Input: python predict.py Spain Germany

  ①  LOAD DATA
  │   4,200+ matches, 201 rankings, 47,000+ goals
  │
  ②  GET LAST 12 MATCHES (Form)
  │   Weighted by time decay, tournament importance, opponent strength
  │
  ③  BAYESIAN SHRINKAGE (k=9.4)
  │   Pull toward global average (1.36)
  │
  ④  FIFA RANKINGS
  │   Log curve → ranking_factor
  │   Blend: 40% ranking + 60% form (both top-30)
  │          25% ranking + 75% form (others)
  │
  ⑤  STAR PLAYERS (relative)
  │   21 nations, 50+ players
  │   Dampened (0.65x), capped (1.35x max)
  │
  ⑥  COACHES IMPACT (NEW v8)
  │   Layer 1: Coach ATK mult (own goals boosted)
  │   Layer 2: Coach DEF mult (opponent goals reduced)
  │   Layer 3: Tier gap bonus (if >= 2 tier difference)
  │   Dampened (0.75x) for realism
  │
  ⑦  HOME ADVANTAGE
  │   Home: x1.06 | Away: x0.94
  │
  ⑧  RATIO CAP (top-30 only)
  │   Max ratio = 1.35x between top teams
  │
  ⑨  LAMBDA FLOOR
  │   Minimum 0.25 xG (no team scores 0.0)
  │
  ⑩  DIXON-COLES SCORE MATRIX
  │   Poisson probabilities + low-score correlation (rho = -0.04)
  │   Normalized 9x9 matrix
  │
  ⑪  MONTE CARLO SIMULATION (10,200 matches)
  │   Sample from DC matrix (not plain Poisson)
  │   Analytical and simulation results cross-validate
  │
  ⑫  OUTPUT
      Win/Draw/Loss percentages
      Expected goals (lambda)
      Top scorelines with probabilities
      Star player data
      Coaches data (name, tier, style, impact, honors)
      Match type classification
─────────────────────────────────────────────────────────────
